Overview

What this project presents

The paper introduces an indoor mixed-reality testbed that supports the simultaneous coexistence of real and simulated drones in the same experimental environment.

Abstract

This paper presents a mixed-reality indoor testbed for multi-drone experiments with real and simulated agents. The setup extends a previous mixed-reality indoor arena by enabling their simultaneous coexistence within the same environment. It integrates Crazyflie drones, an OptiTrack motion capture system for pose measurement of real agents, an object-oriented MATLAB framework for dynamic simulation and implementation of guidance and control algorithms, a Python-based communication interface for command transmission to the real drones, and a Unity 3D visualizer for integrated visualization of all agents. The testbed is demonstrated in three scenarios: a fully simulated case with three drones, a hybrid case with one real and one simulated drone, and a hybrid case with two real and one simulated drone, showing feasibility and flexibility for mixed-reality multi-agent experiments.

Architecture

Hardware and software pipeline

The system combines a motion capture system, radio communication, real drones, dynamic simulation, and 3D visualization in a unified mixed-reality framework.

Motion capture

OptiTrack provides accurate position and attitude estimates for the real drones and streams the rigid-body poses over the local network.

Central coordination

MATLAB acts as the central coordinator, managing setup configuration, simulation routines, and guidance and control algorithms.

Communication and visualization

A Python interface using Crazyswarm handles command transmission to the Crazyflie drones, while Unity 3D displays all drones and obstacles in one environment.
